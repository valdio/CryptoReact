## How to run the app?

```
git clone repo

cd ProjectFolder/
```

Start the JS server

```
react-native start
```

Run the app

```
react-native run-android

//or
react-native run-ios
```


Use the logs
```
react-native log-android

//or
react-native log-ios
```
