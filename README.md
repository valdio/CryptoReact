### NOTE! 
This app is used as a starter React Native app to quickly set up a new project.  

Used libraries:
* redux
* react-redux
* redux-logger
* redux-thunk
* redux-persist
* react-router-native

For testing 
*  jest
* enzyme


## How to run the app?

```
git clone repo

cd ProjectFolder/
```

Start the JS server

```
react-native start
```

Run the app

```
react-native run-android

//or
react-native run-ios
```


Use the logs
```
react-native log-android

//or
react-native log-ios
```
